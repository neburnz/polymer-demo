polymer-demo
============

# Polymer proof of concept

## Requirements:
* Bower (http://bower.io/)

## Instructions:

1. Clone the project
2. Download Polymer components
  Polymer:

    bower install --save Polymer/polymer

  Core:

    bower install --save Polymer/core-elements

  Paper:

    bower install --save Polymer/paper-elements

  Google Maps:

    bower install --save google-map
